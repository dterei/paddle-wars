This is a paddle wars game, or classically known as 'Pong.'

The game is written in the Java programming language, and is an Applet.

It was written, and is still currently been written as part of a Year 11, Software Design and Development assignment at Barker College, Hornsby, Sydney, Australia, it has been handed in now, but some development will still continue.

The Developers of this program are David Terei, and Chris Bell.
